# 🧠 Advanced To-Do App

An interactive and feature-rich To-Do application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ✅ Add & Delete Tasks  
- ✔️ Mark tasks as completed  
- 💾 Local Storage (data persists after refresh)  
- 📅 Assign dates to tasks  
- 🔍 Search functionality  
- 🎯 Filter tasks (All / Pending / Completed)  
- 🔥 High Priority task marking  
- 🎨 Clean and modern UI  

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (DOM Manipulation)
